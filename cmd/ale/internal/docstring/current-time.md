---
title: "current-time"
description: "returns the current system time (in nanoseconds)"
names: ["current-time"]
usage: "(current-time)"
tags: ["os"]
---

Returns the system's current time, measured in nanoseconds since January 1, 1970 UTC.

#### An Example

```scheme
(current-time)  ;; returns 1554720691499809478
```
